Kiosk Mode Options
Type in name Students can search for their own name to create a pass.

You must have ID numbers set up to turn this off.Admins can set up ID numbers in integrations.
Type in ID number Students can search for their name by typing in their ID number.

You must have ID numbers set up to turn this on.Admins can set up ID numbers in integrations.
ID Scanning Students can scan their ID to create a pass. You must have a compatible third-party scanner attached to this device. Learn more

You must have ID numbers set up to turn this on.Admins can set up ID numbers in integrations.
